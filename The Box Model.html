THE BOX MODEL

The CSS Box Model is generally described as a box that wraps around every HTML element. It consists of : margins, borders, padding and the Actual content.

  • Content: This is the where the text and images appear.
  • Padding: It clears an area around the content. The padding is always transparent.
  • Border: This is a border that goes around the padding and the content.
  • Margin: The margin clears an area outside of the border. It is also transparent.

    • Width and Height

    When the width and height properties of an element are put in place, only the content area is affected.

    Also, to calculate the full size of an element, you must also add the padding, borders, and the margins.
