The CSS Float Property

  • The "Float" Property indicates whether an element should float to the left, right, or not at all. Absolutely positioned elements ignore the float property.
  • When you float elements, the browser takes them out of the regular document flow and places them at the top left or right edge of the containing elememt.

The Clear Property

  • Elements next to a floating element will flow around it. To avoid this, you use the "clear" property. The clear Property resumes the normal document flow. could be clear left, right, or both.
  • Floating Elements can produce very flexible layouts.
