Relative and Absolute Element Positioning

  • These Positioning schemes enable you to specify precise offsets to move the target element to a different part of the page origin.

Static Positioning

  • This is equivalent to the normal document flow. It is a default setting for all elements except html.

Relative Positioning

  • When this is applied, the element is positioned relative to its position in the normal document flow. Therefore, if offsets are applied on that element, there will be offset from the original document flow position of that elememt.
  • The Positioning offset properties are : top, bottom, right, and left.
  • The element is NOT taken out of normal document flow. If moved, its original spot is preserved.

Absolute Positioning

  • Here, All offsets(top, bottom, right, left) are relative to the position of the nearest ancestor which has the positioning set on it, other than static.
  • The element is taken out of its normal document flow.
  • If the Container element is offset, everything inside is offset with it.
