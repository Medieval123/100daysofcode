The Javascript 'Event' Argument.

Say Hello to:

The 'event' argument is an argument that is passed by the Javascript Engine into every single event handler.

The Mozilla Development Network talks about all kinds of events that can happen and obviously, depending on what type of event that happens, either a key press, or a mouse press/mousemove which is in interaction with the mouse.

There are different types of objects that can contain different types of properties related to an event type. Some of them are:

  • KeyboardEvent object: This is related to keyboard events.
  • mouseEvent object: gives you a bunch of different properties of the mouse event passed like the clientX and clientY.
  • clientX is the X coordinate of the mouse pointer in the local DOM content coordinates. Types of mouse events include:

    • mouse out.
    • mouse over.
    • mouse move.
