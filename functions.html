FUNCTIONS

A Javascript function is a a block of code designed to perform a particular task. A Function is executed when "something" invokes/calls it.

  • A Javascript function is defined with the function keyword, followed by the name, and then Parentheses ().
  • Functions in Javascript are first class data types; meaning that whatever you could do to a variable or an object, you could also do to a function.
  • Functions in Js are also objects.
  • Every object has this inherent method or function called "to strings".

A Function is much the same as a Procedure or a Subroutine, in other Proramming Languages.
