OBJECT CREATION

  • Once we create a reference to an object, we are ready to create properties of that object.
  • In Javascript, when a property doesn't exist and you simply mention that property in an object, Javascript engine creates that property for you.

The Properties are assigned with:

  • The dot Notation.
  • The bracket Notation.
  • The dot Notation cannot always be used because it will only work with whatever is a valid Javascript identifier, or a valid function/Variable name.
