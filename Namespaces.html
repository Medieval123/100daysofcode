Javascript Namespaces.

A Namespace is defined as a declarative region that provides a scope to the identifiers(the names of types, functions, variables, etc) inside it.

It is well known that other languages use namespaces,where multiple namespace blocks with the same name are allowed.

We can do that too in Javascript, if we fake it.

How Namespaces are Created.

  • We start by creating two variables with an empty value {},
  • We use the dot operator to assign greet to the variable,
  • And Boom!, we have our Namespace.
