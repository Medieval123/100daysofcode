Function Constructors, Prototype, and the 'this' keyword.

With Function constructors, you can create an object using the keyword 'new'.

What happens when you invoke a function?

  • A New Execution context is created within which whatever is inside the function is actually executed.
  • When you invoke a function together with the 'New' keyword, the Js engine makes "this" point to the object itself that got created.

"This"

  • In Javascript, the "this" keyword refers to an object.
  • It points to the global window object. That is, the object that has all the parameters and properties in Javascript.
