Immediately invoked function expressions(IIFE's)

What are Immediately Invoked function Expressions?

IIFE's are functions that are executed immediately after being defined.

Any function expression can be a IIFE by wrapping it in parentheses, and adding a following pair of parentheses at the end.

Syntax

(function() {
//Code that runs in your function }}();

When to Use an IIFE?

  • Aliasing global variables.
  • Creating private variables and functions.
  • Asynchronous functions in loops.
