CSS Media Queries

  • Media Queries enable you to group styles together and target them to devices (medium or large) based on some certain criteria. Each media feature results to either true or false. If true, the style within the curly braces apply.

Basic Syntax of a media query

  • @media (media feature)
  • @media (media feature) logical operator (media feature)

media features can be combined using logical operators(OR & AND).

  • Usually, you provide base styling, then change or add to them in each media query.
  • NB: Be careful not to overlap range boundaries(breakpoints).
